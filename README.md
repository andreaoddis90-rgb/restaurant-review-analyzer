# 🍽️ Restaurant Review Analyzer

An AI-powered web app that analyzes restaurant reviews and provides 
actionable insights through interactive dashboards.

## 🚀 Features
- AI sentiment analysis (positive/negative/neutral)
- Score rating from 1 to 10 per review
- Category detection (food, service, ambiance, value)
- Interactive pie and bar charts
- Detailed review breakdown table

## 🛠️ Built With
- Python 3.13
- Streamlit
- Anthropic Claude API
- Pandas
- Plotly

## 💡 How to Run
1. Clone this repository
2. Install dependencies: `pip install streamlit anthropic pandas plotly`
3. Add your Anthropic API key in app.py
4. Run: `streamlit run app.py`
